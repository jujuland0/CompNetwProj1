How does my domain resolve to an IP address ?

  1. When my domain is entered into a browser, for speed it first checks the cache in case the domain has already been resolved recently. If the IP is already here, it is used.
  2. If the browser does not have the IP stored in cache, the OS DNS cache is checked.
    • Here, I have checked the OS DNS cache using nslookup.
  3. Finally, it asks a recursive resolver for the IP address, which will be provided by either the ISP of the user, Google DNS, or Cloudflare DNS.
    • Here, because I am hosting on github, I am suing their DNS servers.
