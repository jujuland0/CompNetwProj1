My site has both an IPv4 and IPv6 address. This is so that both can be reached by a client device.

Usually IPv6 will be chosen but if not IPv4 will be used automatically. In essence, this ensures compatibility with old and new, mobile and desktop devices.

My site already has IPv6 support due to the use of GitHub Pages, which supports both IPv4 and IPv6.
